# A simple bot for AI game Halite

Learn more on https://halite.io

# Setup
Install the game engine (just once)
````
sh -c "$(curl -fsSL https://raw.githubusercontent.com/HaliteChallenge/Halite/master/environment/install.sh)"
````

# Build and Run
````
./runGame.sh
````
